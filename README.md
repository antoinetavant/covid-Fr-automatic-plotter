# covid-Fr-automatic-plotter

Fesh the Covid Data and plot some graphs, in order to obtaine quickly the important figures

# Sources
The data comes from the french gouvernement. It is updated daily but there is a delay from the latest data to be added. Curently it is around 5 days.

[link to the web page](https://www.data.gouv.fr/fr/datasets/donnees-relatives-aux-resultats-des-tests-virologiques-covid-19/)

# tools

I used pandas for data managment, ant matplotlib to generate the figure.

I run the script daily in oerder to update [this page of my blog](https://antoinetavant.github.io/post/covid/)
